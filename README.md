## Решение проблем с базой данных

При возникновении ошибок с базой данных выполните следующие шаги:

### Вариант 1: Downgrade MySQL

Выполните downgrade до версии **MySQL 5.7.x**.

### Вариант 2: Настройка конфигурации

1. Откройте файл конфигурации MySQL:
   - **Windows**: `my.ini`
   - **Linux**: `my.cnf`
   
   > **Примечание**: Для OpenServer и других локальных серверов путь к файлу может отличаться. Рекомендуется найти актуальный путь в Интернете.

2. Добавьте следующие строки в секцию `[mysqld]`:

```ini
innodb_file_per_table = 1
innodb_strict_mode = 0
```

3. Перезапустите сервер MySQL.

---

Эти действия должны исправить возможные проблемы с импортом базы данных и запуском сервера.
